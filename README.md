## Tech Stack

- PHP 8.3
- Composer 2.7.6
- Node 20.14.0
- NPM 10.7.0
- Vue 3
- Tailwind CSS 3.4.4
- Inertia

## Steps to run the project

``git clone the project``

``cd/project``

``create .env``

``composer install``

``npm install``

``npm run dev``

``php artisan serve``

## Seed data

``php artisan migrate:fresh --seed``
